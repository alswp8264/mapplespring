# Mapple Spring 요구사항 명세서

> 메이플스토리 캐릭터 정보 조회 · 커뮤니티 웹 서비스
> 작성일: 2026-06-30 · 기준: 현재 구현 완료 상태

---

## 1. 문서 개요

### 1.1 목적
본 문서는 "Mapple Spring" 웹 서비스에 **실제 구현된 기능**을 정리한 요구사항 명세서이다. 넥슨 메이플스토리 Open API를 기반으로 캐릭터 정보 조회·랭킹·코디(룩북)·커뮤니티 기능을 제공하며, 일반/소셜/얼굴 인식 로그인을 지원한다.

### 1.2 시스템 범위
- 메이플스토리 캐릭터 검색 및 상세 정보(스탯·장비·코디·유니온·아티팩트) 조회
- 종합/유니온/무릉도장 등 각종 랭킹 조회
- 이벤트·캐시샵 공지, 룩북, 코디 분석, 게시판/댓글 커뮤니티
- 회원 관리 및 3종 로그인(일반 / 소셜 OAuth / 얼굴 인식)

### 1.3 사용자 정의
| 구분 | 설명 |
|---|---|
| 비회원(Guest) | 검색·랭킹·룩북·게시판 등 **열람** 가능 |
| 회원(Member) | 즐겨찾기, 스펙 히스토리, 룩북 등록/좋아요, 글/댓글 작성, 얼굴 등록 |
| 관리자(Admin) | 회원 목록 관리 등 관리 기능 |

## 2. 시스템 환경 및 기술 스택

| 구분 | 내용 |
|---|---|
| 언어/런타임 | Java 21 |
| 프레임워크 | Spring MVC (XML 설정: `web.xml`, `dispatcher-servlet.xml`, `applicationContext.xml`) |
| 영속성 | MyBatis (게시판·회원·룩북·즐겨찾기 등) + Spring Data JPA / Hibernate (유니온 계정·캐릭터) |
| DB | Oracle XE (`jdbc:oracle:thin:@localhost:1521:xe`, ojdbc11) |
| View | JSP + JSTL (Jakarta Tags) |
| JSON | Jackson |
| WAS | Apache Tomcat 11 (Jakarta EE) — 컨텍스트 경로 `/Mapple_Spring`, 포트 8088 |
| 외부 API | 넥슨 메이플스토리 Open API, Google OAuth2, Kakao OAuth2 |
| 얼굴 인식 서버 | Python Flask + `face_recognition`(dlib) — 별도 프로세스, 포트 5001 |
| 접속 주소 | http://localhost:8088/Mapple_Spring/ |

## 4. 비기능 요구사항

| 구분 | 내용 |
|---|---|
| 성능/캐싱 | 인기 캐릭터 이미지·공지 썸네일을 캐싱(메모리/DB)하여 외부 API 재호출 최소화 |
| 레이트리밋 대응 | 넥슨 API 호출 순서를 **핵심 데이터(랭킹) 우선 → 이미지 보강 후순위**로 분리하고, 429 발생 시 재시도(백오프) 및 호출 간 텀 적용 |
| 보안 — 인증 | OAuth는 표준 Authorization Code 흐름(비밀번호 미수집), 얼굴 로그인은 서버 측 대조 |
| 보안 — 얼굴 | 거리 임계값 0.40 + 다중 얼굴 거부로 타인 오인증 방지 |
| 가용성 | 외부 API 오류 시 화면 깨짐 없이 안내 메시지/플레이스홀더로 우아하게 처리 |

## 7. 제약사항

- 넥슨 Open API는 **하루 지연 데이터**를 기준으로 하며, 최근 30일 내 접속 캐릭터만 조회 가능
- 현재 **개발 단계 API 키** 사용 → 일부 엔드포인트 접근 제한(8장 참조)
- 로컬 환경(`localhost:8088`) 기준 — 외부 사용자가 접속하려면 별도 공개(배포/터널링) 필요

## 8. 미구현 / 보류 기능 (외부 권한·정책 이슈)

> 코드 구현은 되어 있으나 **외부 서비스의 권한/등급 정책** 때문에 비활성화한 항목.

| 기능 | 사유 | 해소 조건 |
|---|---|---|
| 헥사(6차) 스킬 탭 | 넥슨 **개발 단계 키**로 `hexamatrix` 호출 시 403 Access Denied | 넥슨 앱 정식 전환 후 권한 보유 키로 교체 |
| V매트릭스(5차) 탭 | 동일 — `vmatrix` 403 Access Denied | 동일 |
| 본캐/부캐 탭 | 넥슨 API가 임의 계정의 부캐 닉네임을 제공하지 않음(프라이버시) | 구조적 불가(대체 데이터원 필요) |
| 카카오 — 타 계정 로그인 | 카카오 앱이 **개발용**이라 팀원 외 계정 로그인 거부 | 동의항목 정리(닉네임만) 또는 **비즈앱 전환** |
